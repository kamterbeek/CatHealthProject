# Cat Health Analytics

A personal data engineering and analytics project that combines:

- Petlibro feeder telemetry
- Petlibro fountain telemetry
- Manual stool observations
- Weight measurements
- Health notes

to monitor long-term feline health trends.

## Goals

1. Track feeding behavior over time.
2. Track hydration patterns.
3. Analyze stool quality and frequency.
4. Detect potential health anomalies.
5. Build dashboards and alerts.
6. Practice data engineering and analytics skills.

## Technologies

- Python
- Pandas
- PostgreSQL
- SQL
- Streamlit
- Docker
